## 0. One-time host fix (do this first)

The ~40 "errors" in your terminal were **skips, not failures** — Testcontainers couldn't find a Docker-compatible socket because the rootless Podman API socket wasn't running. Enable it once and it survives reboots:

```bash
systemctl --user enable --now podman.socket
```

The new `ContainerEngineDiscovery.cs` then finds `$XDG_RUNTIME_DIR/podman/podman.sock` automatically (honoring `DOCKER_HOST` if you've set one) and disables Ryuk, which rootless Podman doesn't need.
